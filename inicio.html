Proyecto de ciclo

TÍTULO DEL PROYECTO

Aplicación móvil para ayudar a personas con discapacidad visual en la compra


NOMBRE PROVISIONAL DE LA APP

Blinder


INTRODUCCIÓN

La discapacidad visual puede ser una gran barrera para realizar tareas cotidianas, como ir de compras. Sin embargo, con la tecnología actual, es posible desarrollar una aplicación móvil que ayude a estas personas a superar esta barrera y realizar la compra de manera autónoma y segura.


OBJETIVO

El objetivo principal de esta aplicación es proporcionar a las personas con discapacidad visual una herramienta útil para facilitar el proceso de compra en cualquier establecimiento. La aplicación debería incluir funciones como la lectura en voz alta de los productos, la descripción de los mismos, así como la posibilidad de crear listas de la compra y navegar por el establecimiento de manera autónoma.


FUNCIONALIDADES

    Escaneo de códigos de barras: la aplicación incluiría una función de escaneo de códigos de barras para que los usuarios puedan escanear los productos y obtener información sobre ellos, como el nombre, precio y descripción.
    Lectura en voz alta: la aplicación incluiría una función de lectura en voz alta para que los usuarios puedan escuchar la información sobre los productos, incluyendo el nombre, precio y descripción.
    Creación de listas de la compra: los usuarios podrían crear listas de la compra de manera previa a su visita al establecimiento, con la posibilidad de añadir productos de manera autonóma.
    Navegación en el establecimiento: la aplicación incluiría una función de navegación que permitiría a los usuarios encontrar los productos en el establecimiento de manera autónoma mediante la localización geográfica en el establecimiento o mediante la navegación por categorías de productos.
    Interfaz de usuario accesible: la aplicación incluiría una interfaz de usuario accesible para que las personas con discapacidad visual puedan utilizarla con facilidad.

IMPLEMENTACIÓN

La aplicación se desarrollaría utilizando tecnologías móviles como Android o iOS y se integraría con un sistema de escaneo de códigos de barras y un sistema de navegación. Para implementar una funcionalidad de localización geográfica en un supermercado, se podrían seguir los siguientes pasos:
1. Diseño del mapa: se debería crear un mapa del supermercado con las diferentes secciones y estanterías. Este mapa se utilizaría como base para la navegación en la aplicación.
2. Integración de la tecnología de localización: se debería integrar tecnologías de localización, como GPS, Bluetooth o iBeacons, en el supermercado para permitir la localización precisa de los usuarios en la aplicación.
3. Creación de un sistema de navegación: se debería crear un sistema de navegación que utilice la información del mapa y la localización del usuario para guiarlo a través del supermercado. Esto podría incluir indicaciones visuales y de voz para ayudar al usuario a encontrar los productos deseados.
4. Integración con la aplicación: se debería integrar el sistema de navegación con la aplicación móvil para que los usuarios puedan acceder a él mediante la aplicación.
5. Pruebas y ajustes: Finalmente, se realizarían pruebas en el supermercado para asegurar que el sistema funciona correctamente y se realizarían ajustes si es necesario para mejorar la experiencia del usuario.
Es importante tener en cuenta que para implementar la localización geográfica en un supermercado, se requiere una inversión en tecnología y en el desarrollo de la aplicación, así como una buena planificación y colaboración entre los diferentes departamentos y equipos involucrados.


iBeacons

Para implementar una funcionalidad de localización geográfica en un supermercado utilizando iBeacons, se podrían seguir los siguientes pasos: Adquisición de iBeacons: se deberían adquirir iBeacons y configurarlos para transmitir información a través de Bluetooth. Estos iBeacons se colocarían en diferentes puntos del supermercado, como estanterías y pasillos. Integración de iBeacons con el sistema: se debería integrar el sistema de iBeacons con el sistema de navegación del supermercado, para que los usuarios puedan recibir información sobre su ubicación en el supermercado mediante la aplicación. Desarrollo de la aplicación: se debería desarrollar una aplicación móvil que permita a los usuarios escanear los iBeacons y recibir información sobre su ubicación en el supermercado. Pruebas y ajustes: se realizarían pruebas en el supermercado para asegurar que el sistema funciona correctamente y se realizarían ajustes si es necesario para mejorar la experiencia del usuario. Capacitación del personal: se debería capacitar al personal del supermercado para que puedan ayudar a los usuarios a utilizar el sistema de navegación y responder preguntas sobre el mismo. Es importante tener en cuenta que los iBeacons requieren de baterías y una configuración adecuada para poder funcionar correctamente, así como una buena planificación de la colocación de estos dispositivos para cubrir todo el supermercado y permitir una navegación precisa. Asimismo, es necesario contar con una aplicación compatible con la tecnología iBeacon para poder recibir la información de estos dispositivos.
